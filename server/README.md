## Create a .env file in server folder:

```sh
# App data
HOST = "localhost"
PORT = 3000

# Mysql connection
MYSQL_HOST = "localhost"
MYSQL_PORT = "3306"
MYSQL_USER = ""
MYSQL_PASSWORD = ""
MYSQL_DATABASE = "fullstackreactapp"

# Email connection
EMAIL_HOST = "smtp.gmail.com"
EMAIL_PORT = 465
EMAIL_SECURE = true
EMAIL_ADDRESS = ""
EMAIL_PASSWORD = ""
```
